# Claims

Claim 1: Visible geometry is an incomplete state representation for assembly manipulation when hidden fixtures constrain motion.

Claim 2: Probe outcome residuals can identify a discrete latent fixture mode well enough to improve the next manipulation action.

Claim 3: In the deterministic toy benchmark, latent-fixture reasoning succeeds in 1995/2000 seeds with 5 collisions, compared with geometry-only 621/2000 successes and 1379 collisions.

Claim 4: The useful abstraction is not simply uncertainty; it is an interpretable hidden cause connected to a release action.
